# ⚡ Mohan Krishna - Neo-Brutalist Portfolio

![Portfolio Preview](https://github.com/mohankrishna-9704/Mohan-Portfolio/raw/main/public/preview-banner.png)

A cutting-edge, high-performance portfolio built with a **Neo-Brutalist** aesthetic. This project combines raw architectural design with fluid high-end animations and interactive components.

## 🚀 Live Demo
**[mohan-portfolio.vercel.app](https://mohan-portfolio.vercel.app)** *(Deploying soon...)*

## ✨ Features

### 🛠️ Interactive Experience
- **Draggable Stickers**: Pick up and move tech icons across the screen. Custom SVG stickers that feel alive.
- **Interactive Terminal**: A fully functional CLI mode to explore the portfolio via typed commands.
- **Command Palette**: Quickly navigate the site or trigger actions with `Cmd/Ctrl + K`.
- **Custom Audio Engine**: Contextual sound effects for clicks, hover states, and theme switches.

### 🎨 Design System
- **Neo-Brutalist UI**: Aggressive shadows, thick borders, and vibrant color palettes (Acid Lime & Brutal Red).
- **Dark Mode Support**: Seamless transition between light and dark themes with system preference detection.
- **Fluid Animations**: Powered by Framer Motion for high-frame-rate transitions and scroll reveals.
- **Custom Cursor**: Reactive cursor that interacts with elements and changes state based on hover targets.

### 📊 Content Sections
- **Dynamic Stats**: Interactive counters showing tech exposure and experience.
- **Project Carousel**: Touch-enabled, snap-mandatory horizontal slider for mobile users.
- **Skill Marquee**: INFINITE scrolling tech marquee for high-density skill display.
- **Education & Journey**: A timeline-based walkthrough of career milestones.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion 12](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohankrishna-9704/Mohan-Portfolio.git
   cd Mohan-Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
